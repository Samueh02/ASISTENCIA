📖 Instrucciones

📌 Añadir asignatura: Escribe el nombre de la asignatura y el porcentaje mínimo de asistencia requerido, luego pulsa "Añadir".

📌 Gestionar asistencia: Usa los botones ➕ y ➖ para registrar asistencias y faltas.

📌 Añadir clase al horario: Selecciona el día, la hora de inicio y fin, introduce el nombre de la clase y pulsa "Añadir Clase".

📌 Editar horario: Doble clic sobre el nombre de una clase para modificarlo.

📌 Eliminar clases: Pasa el cursor sobre una clase y pulsa "✖" para eliminarla individualmente.

📌 Vaciar todo el horario: Usa el botón "Vaciar Horario" si deseas reiniciar el horario completamente.
